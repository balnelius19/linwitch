╚════════════════════════════╝

Bienvenue dans le dossier "001 Grimoire" du Bureau.

Ce projet est une tentative expérimentale de création
d’un environnement magique accessible via une interface
simple en script Bash.

🎯 Objectifs :
- Proposer une interface ludique (type menu rôliste)
- Offrir des commandes et scripts pour débutants
- Créer une version exportable en mode "distro magique"

📁 Contenu :
- 001-grimoire.sh : script principal
- aide_des_sorts.txt : document d’aide et d’exemples
- icônes/ : symboles et illustrations (optionnel)
- grimoire.desktop : raccourci graphique (créé à la fin)

🔮 Instructions :
Lancez le script 001-grimoire.sh pour commencer.
Il est recommandé de tester l’environnement sur un live system
avant exportation finale.

📦 Distribution :
À copier dans /etc/skel/Magie/ pour intégration automatique
dans l’environnement utilisateur post-installation.
